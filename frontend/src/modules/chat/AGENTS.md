# Agente: Asistente Virtual (Chatbot)

## Descripción
Este agente actúa como un asistente virtual interactivo para los asistentes al LKS Tech Day. Su objetivo principal es resolver dudas y demostrar capacidades de IA conversacional (simulada o real).

## Archivos Principales
*   **Componente Principal:** `Chat.jsx`

## Funcionalidades
*   **Chat Conversacional:** Interfaz tipo chat con burbujas de mensaje para usuario y bot.
*   **Respuestas Predefinidas (Mock):** Detecta palabras clave como "hola", "tech day", "ia", "lks" para dar respuestas contextuales.
*   **Simulación de Escritura:** Indicador visual de "Escribiendo..." para mayor realismo.
*   **Persistencia (Sesión):** Mantiene el historial de la conversación durante la sesión (estado React).
